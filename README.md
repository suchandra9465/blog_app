# blog_app
Blog App with Fast API and MongoDB


To run the project in your local

1) First clome the project 

    git clone https://github.com/suchandra9465/blog_app.git
    
    
2) Install all the modules and prerequisite from requirements.txt

   pip install requirements.txt

3) Once done use the command to run the app

  python app/main.py

4) Now open http://localhost:8000/docs and checkout all the api's implemented
